Start dates
Sharing rule: Partners share one cohort instance when they have the same SKU, same start date, and same early access date. Early access date = previous cohort start date for that partner, which changes monthly. Citizens Bank often gets its own instance because their biweekly cadence means a different EA date than monthly partners. Pearson's T-15 deadline is handled at the program cohort level — it does not affect sharing eligibility.
